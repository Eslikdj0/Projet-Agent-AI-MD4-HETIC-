EXAMPLE 3: Full FAISS indexation pipeline
───────────────────────────────────────────
from indexation import DataIndexer
from sentence_transformers import SentenceTransformer
import faiss
import numpy as np

# 1. Load chunks
indexer = DataIndexer("data/chunks/chunks.json")
chunks = indexer.load()
texts = indexer.get_texts()

# 2. Create embeddings
model = SentenceTransformer("paraphrase-multilingual-mpnet-base-v2")
embeddings = model.encode(texts)

# 3. Build FAISS index
index = faiss.IndexFlatL2(768)
index.add(np.array(embeddings, dtype=np.float32))

print(f"✓ FAISS index created: {index.ntotal} vectors indexed")

# 4. Search example
query = "Movie with action scenes"
query_embedding = model.encode([query])[0]
distances, indices = index.search(np.array([query_embedding], dtype=np.float32), k=5)
print(f"✓ Found {len(indices[0])} similar chunks")

================================================================================
API REFERENCE
================================================================================

CLASS: DataIndexer

Constructor:
  DataIndexer(data_path)
    Parameters:
      data_path (str) - Path to JSON or CSV file
    Example: indexer = DataIndexer("data/chunks/chunks.json")

Methods:

  load() → list or dict
    Description: Load data from file
    Returns: Loaded data (list for JSON arrays, dict for objects)
    Example:
      data = indexer.load()
      print(f"Loaded {len(data)} items")

  get_texts() → list of str
    Description: Extract text content for embedding
    Returns: List of text strings
    Example:
      texts = indexer.get_texts()
      for text in texts[:3]:
          print(text[:100])

  index() → dict
    Description: Prepare data for indexation
    Returns: Dictionary with keys:
      - 'count' (int): Number of items
      - 'texts' (list): Text strings for embedding
      - 'data' (list/dict): Original data records
    Example:
      indexed = indexer.index()
      print(f"Count: {indexed['count']}")
      print(f"First text: {indexed['texts'][0]}")
      print(f"First data: {indexed['data'][0]}")


================================================================================
DATA FORMATS SUPPORTED
================================================================================

✓ JSON files with arrays
  Format: [{"field": "value"}, {"field": "value"}]
  Auto-detection: Looks for 'text' or 'title' field

✓ JSON files with objects
  Format: {"key": {"field": "value"}}
  Converted to dict for processing

✓ CSV files
  Format: CSV with headers
  Auto-converted to records

================================================================================
TROUBLESHOOTING
================================================================================

Q: FileNotFoundError: "data/chunks/chunks.json" not found
A: Ensure you're running from the project root directory
   Current working directory should be: Projet-Agent-AI-MD4-HETIC-

Q: No 'text' field found
A: Check that the JSON file has expected structure
   Run: indexer.load() first to inspect data

Q: Memory error with embeddings
A: Process in batches instead of all at once
   embeddings = model.encode(texts, batch_size=32)
